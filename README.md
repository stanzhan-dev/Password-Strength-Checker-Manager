# Password Strength Checker & Manager

A simple Python GUI app to **check password strength**, **generate strong passwords**, and **keep a history** of the last 5 passwords youâ€™ve used.

![App Screenshot](screenshot.jpg)

---

## Features

- Check the strength of any password and get feedback  
- Generate random strong passwords  
- Copy passwords to clipboard easily  
- Keep a masked history of the last 5 passwords
- Toggle history visibility (show/hide)  
- Export password history to a text file  

---

## Requirements

- Python 3.7+  
- Tkinter (built-in with Python)  

No extra packages required.
